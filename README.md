# Mock API Service for Unit Testing in SwiftUI ViewModels

A SwiftUI-friendly mock API service setup to simplify unit testing of ViewModels. This project demonstrates how to inject mock APIs into your SwiftUI architecture for reliable and fast unit tests — without depending on real network calls. Perfect for apps following MVVM.

## Features

* Mock API responses for SwiftUI ViewModels
* Supports success & failure scenarios
* Easy injection into ViewModels
* Helps write clean, deterministic unit tests
